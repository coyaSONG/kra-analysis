# 경마 예측 v7.0

경주 데이터를 분석하여 1-3위 예상 말 3마리를 선택하세요.

## 분석 기준
1. **기본 전략**: win_odds 낮은 순 (시장 평가 신뢰)
2. **선택 방법**: 
   - 1-2위 인기마는 기본 포함
   - 3번째는 3-5위 중에서 선택
   - 단, 체중 변화 ±20kg 이상이면 제외 고려
3. **필수 제외**: win_odds가 0인 말 (기권/제외)
