# data.go.kr API 활용신청 브라우저 자동화 프롬프트

아래 프롬프트를 브라우저 조작 에이전트에게 전달하세요.

---

## 프롬프트

```
# 작업: data.go.kr 한국마사회 OpenAPI 활용신청

## 목표
data.go.kr에서 아래 8개 한국마사회 OpenAPI에 대해 활용신청을 완료한다.
이미 로그인된 상태라면 바로 신청을 시작하고, 아닌 경우 로그인부터 진행한다.

## 사전 조건
- 사이트: https://www.data.go.kr
- 계정이 이미 존재하고 로그인 가능한 상태
- 기존에 아래 4개 API는 이미 신청 완료됨 (중복 신청하지 말 것):
  - API214_1 경주상세결과
  - API8_2 경주마정보
  - API12_1 기수정보
  - API19_1 조교사정보

## 신청 대상 API 목록 (8개)

우선순위 순서대로 진행:

### 1. 주로정보 (Track)
- 검색어: "한국마사회 경주로정보" 또는 "한국마사회 주로"
- API ID: API189_1
- endpoint: /B551015/API189_1/Track_1
- 용도: 주로상태(함수율, 기온, 풍속 등) 수집

### 2. 통합배당정보
- 검색어: "한국마사회 통합배당" 또는 "한국마사회 배당"
- API ID: API160_1
- endpoint: /B551015/API160_1/integratedInfo_1
- 용도: 단승/복승/연승 등 통합 배당률 수집

### 3. 경주계획
- 검색어: "한국마사회 경주계획"
- API ID: API72_2
- endpoint: /B551015/API72_2/racePlan_2
- 용도: 사전 경주 편성 정보(거리, 조건, 등급 등) 수집

### 4. 출전취소정보
- 검색어: "한국마사회 출전취소" 또는 "한국마사회 출마취소"
- API ID: API9_1
- endpoint: /B551015/API9_1/raceHorseCancelInfo_1
- 용도: 기권/제외마 자동 필터링

### 5. 기수별 경기 성적
- 검색어: "한국마사회 기수성적" 또는 "한국마사회 기수별"
- API ID: API11_1
- endpoint: /B551015/API11_1/jockeyResult_1
- 용도: 기수 누적 성적 통계

### 6. 마주정보
- 검색어: "한국마사회 마주" 또는 "한국마사회 말소유자"
- API ID: API14_1
- endpoint: /B551015/API14_1/horseOwnerInfo_1
- 용도: 마주(소유자) 정보 수집

### 7. 당일 배당률 종합
- 검색어: "한국마사회 당일배당" 또는 "한국마사회 배당률종합"
- API ID: API301
- endpoint: /B551015/API301/Dividend_rate_total
- 용도: 당일 전체 경주 배당 종합 데이터

### 8. 서울 발주훈련 (발마시험/게이트훈련)
- 검색어: "한국마사회 발주훈련" 또는 "한국마사회 게이트"
- API ID: API329
- endpoint: /B551015/API329/textDataSeGtscol
- 용도: 서울경마장 발주훈련 기록

## 각 API별 신청 절차

각 API에 대해 아래 단계를 반복:

1. https://www.data.go.kr 상단 검색창에 해당 검색어 입력
2. 검색 결과에서 "오픈API" 탭 선택
3. 해당 API를 찾아 상세 페이지 진입
4. "활용신청" 버튼 클릭
5. 신청 양식 작성:
   - 활용목적: "경마 데이터 분석 및 예측 모델 연구"
   - 활용기간: 최대 기간 선택 (보통 2년)
   - 트래픽: "일반" 또는 기본값
   - 기타 필수 항목은 적절히 입력
6. 이용약관 동의 후 신청 완료
7. 신청 결과 확인 (즉시승인 또는 심사대기)

## 주의사항
- 검색어로 API를 찾지 못하면 data.go.kr에서 "한국마사회"로 넓게 검색 후 오픈API 목록에서 찾는다
- 이미 신청된 API는 "활용신청" 대신 "활용중" 또는 "신청완료"로 표시될 수 있음 → 건너뛴다
- 신청 양식은 API마다 약간 다를 수 있으므로 유연하게 대응한다
- 각 API 신청 완료 후 결과(즉시승인/심사대기)를 기록한다
```
